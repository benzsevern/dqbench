# DQBench

The standard benchmark for data quality and validation tools.

The ImageNet of data quality — standardized benchmarks with three difficulty tiers, ground truth for every planted issue, fair scoring on recall and precision, and a single DQBench Score (0-100) for easy comparison.

## Install

```bash
pip install dqbench
```

## Quick Start

```bash
dqbench run --tool your_tool
```

Integrate any tool in about 20 lines of code.
